# Issue tracker

This repo tracks issues in **GitHub Issues** on `a1f/claude-admin`.

Skills like `to-issues`, `to-prd`, `triage`, `diagnose` use the `gh` CLI.

- Create issue: `gh issue create --repo a1f/claude-admin --title "<title>" --body-file <file>`
- View issue: `gh issue view <num> --repo a1f/claude-admin`
- List issues: `gh issue list --repo a1f/claude-admin`
- Add label: `gh issue edit <num> --repo a1f/claude-admin --add-label <label>`
